## Description

A simple API for news searching on the New York Times API and The Guardian API!.

## Installation

```bash
$ npm install
```

## Running the app

```bash

$ npm run start:dev

```
## Built With

- Node.js
- Typescript
- NestJS
- Prettier and Eslint where used, both for linting a formatting the code.
